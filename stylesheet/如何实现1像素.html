    # 如何实现1像素

    # min-resolution

    设备物理像素分辨率与css像素分辨率比值的最小值

    @media screen and (resolution: 2dppx) {
    .class-name {
        height: 0.5px;
    }
}

@media screen and (resolution: 3dppx) {
    .class-name {
        height: 0.5px;
    }
}

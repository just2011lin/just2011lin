        # 可复用性&组合

        # 混入

        1.题目：混入的作用是什么？

        答：通过混入可以将混入对象的选项与组件本身的选项进行组合

        2.题目：使用组件的什么选项来接收混入对象？

        答：mixins

        3.题目：数据对象的混入规则是什么？

        答：递归合并，并以组件数据优先

        4.题目：同名钩子函数的混入规则是什么？

        答：合并为一个数组，并且混入对象的钩子先调用

        5.题目：方法、组件和指令的混入规则是什么？

        答：合并为一个对象并以组件的值为优先。

        6.题目：使用什么方法可以定义全局混入对象？

        答：Vue.mixin方法

        7.题目：怎么设置自定义选项的合并策略？

        答：Vue.config.optionMergeStrategies.myOption = function(toVal, fromVal){ // 返回合并后的对象 }（toVal和fromVal分别是组件还是混入对象的还不清楚）

        8.题目：自定义选项的默认合并策略什么？

        答：覆盖已有的值？（具体怎么覆盖不清楚）

        # 自定义指令

        1.题目：怎么注册一个全局指令？

        答：使用Vue.directive方法并传入指令名和指令选项

        2.题目：怎么注册一个局部指令？

        答：在组件中使用directives选项，并用key作为指令名，value作为指令选项

        3.题目：指令选项中可以定义哪些钩子函数？

        答：bind inserted updated componentUpdated unbind

        4.题目：钩子函数有哪些参数？

        答：el binding vnode oldVnode四个参数，其中binding对象有：name value oldValue expression arg modifiers属性

        5.题目：使用函数简写的方式定义指令时，这个函数会作为什么钩子？

        答：bind update

        # 渲染函数&JSX

        1.题目：在组件选项中渲染函数字段名是什么？

        答：render

        2.题目：createElement方法的参数有哪些，请简述一下？

        答：第一个参数为：组件名或组件选项或resolve值为上述的async函数，第二个参数（可选）：attribute数据对象，第三个参数（可选）：子元素或子组件

        3.题目：createElement方法的第二个参数attribute数据对象有哪些属性？

        答：class style attrs props domProps on nativeOn directives scopedSlots slot key ref refInFor

        4.题目：VNode必须唯一是指什么唯一？

        答：具体来讲，在一个render函数中，一个VNode只能作为一个子元素存在

        5.题目：怎么在渲染函数中实现v-if和v-for

        答：v-if可以用if-else判断来输出不同结果，v-for可以用数组的map方法输出多个VNode

        6.题目：v-model在渲染函数中要怎么实现？

        答：props中接收value，createElement函数的第二个参数中设定domProps的value为组件的prop value,然后在事件处理程序中用$emit出input事件及值

        7.题目：在render函数内事件和按键修饰符怎么处理？

        答：部分修饰符（passive capture once）在事件名前添加前缀来实现，其他修饰符可用事件对象来实现

        8.题目：子组件的render函数内怎么访问插槽和作用域插槽？

        答：使用$slots访问插槽内容，使用$scopedSlots访问作用域插槽

        9.题目：父组件的render函数内怎么设置子组件的插槽内容？

        答：通过createElement第二个参数的scopedSlots选项设置

        10.题目：函数式组件有什么特点？

        答：没有响应式数据也没有生命周期方法

        11.题目：怎么设置一个组件为函数式组件？

        答：在组件选项中设置functional值为true，或在vue文件的template元素上添加functional属性

        12.题目：函数式组件怎么在render函数中接收参数？

        答：通过render函数的第二个参数context接收

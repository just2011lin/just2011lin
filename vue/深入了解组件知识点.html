              # 深入了解组件知识点

              # 组件注册

              1.题目：定义组件名的方式有哪两种？

              答：kebab-case 或 PascalCase

              2.题目：全局注册与局部注册分别是怎么做的（简述即可）？

              答：全局注册可以使用Vue.component方法，局部注册可以用components选项来配置

              3.题目：使用webpack时可以用webpack提供的哪个方法实现基础组件的自动化全局注册？

              答：require.context方法

              # Prop

              1.题目：使用dom模板时，camelCase的prop名需要使用等价的什么命名方式？

              答：kebab-case命名方式

              2.题目：通过将props的值设置为一个对象时可以以key作为prop名、以value为prop值的类型，那么请问可以设置哪些类型？

              答：String Number Boolean Array Object Function Promise等其他构造函数

              3.题目：使用什么方法可以直接将一个对象的所有property都作为prop传给子组件？

              答：v-bind指令不带参数并且用那个对象作为指令值

              4.题目：单向数据流是什么意思？

              答：父级prop的更新会向下传递给子组件，而子组件prop的更新不会。

              5.题目：prop验证对象有哪些选项？

              答：type required default validator

              6.题目：非prop的attribute在子组件中会被如何操作？

              答：默认情况下，class和style属性会与组件根元素上的属性合并，其他属性将替换组件根元素的同名属性

              7.题目：设置了禁止继承attribute后，可以用哪个实例的哪个属性取得传入的attribute？

              答：$attr

              # 自定义事件

              1.题目：事件名推荐使用什么格式来定义？

              答：kebab-case

              2.题目：native修饰符有什么用？

              答：用于修饰事件绑定，会将事件监听绑定在子组件的根元素上

              3.题目：子组件内可以用实例的什么属性获取绑定在这个组件上的所有监听器？

              答：$listeners

              4.题目：v-on="$listeners"的作用是什么？

              答：将所有绑定在这个组件上的监听器添加到指令所在的元素上

              5.题目：怎么对一个prop实现父子组件间的双向绑定？

              答：在父组件中用sync修饰符修饰prop，并在子组件中调用$emit('update:prop', propValue)

              # 插槽

              1.题目：子组件模板中的slot标签内的内容有什么用？

              答：slot标签内的内容将作为默认内容在父组件中未定义slot内容时展示

              2.题目：slot元素上用什么属性名定义具名插槽？

              答：name

              3.题目：v-slot指令中的参数有什么用？

              答：参数为具名插槽中的名字，用于指定在何处使用插槽内容。

              4.题目：怎么将子组件的数据用在父组件的插槽内容中？

              答：子组件中在slot元素内使用v-bind绑定prop名与值，在父组件中使用v-slot命令的值来定义插槽的prop对象名

              5.题目：在使用v-slot指令中没有使用参数，则表示这是用的什么插槽？

              答：默认插槽。

              6.题目：在定义了多个插槽prop后，可以用什么方法方便的取到这些prop？

              答：对v-slot指令的值进行解构来获取其中的prop

              7.题目：具名插槽可以将v-slot指令缩写为什么字符？

              答：#

              8.题目：旧语法slot与slot-scope属性分别有什么用？

              答：slot属性的值为插槽的具名，相当于v-slot指令的参数；slot-scope的值为插槽props的名称，相当于v-slot指令的值

              # 动态组件 & 异步组件

              1.题目：在动态组件外层使用keep-alive元素进行包裹会有什么效果？

              答：会使动态组件在被切换时保持它们原有的状态

              2.题目：使用webpack2和ES2015语法时怎么动态加载组件？

              答：全局注册方法Vue.component的第二个参数和局部注册时的值使用()=>import('脚本相对路径')

              3.题目：异步组件工厂函数怎么处理加载状态？

              答：在()=>import()方法中的import内使用一个配置，配置选项有component、loading、error、delay、timeout

              # 处理边界情况

              1.题目：在组件内部可以用什么实例属性访问根组件和父组件实例？

              答：$root来访问根组件，$parent访问父组件实例

              2.题目：怎么访问子组件实例或子元素？

              答：在html模板中为子组件或子元素设置ref属性并设置值如value，然后可以通过$ref.value来访问子组件或子元素

              3.题目：祖先组件和后代组件分别用什么选项来实现依赖注入？

              答：在祖先组件上使用provide选项来指定提供的数据和方法，在后代组件上使用inject来接收指定的数据和方法。

              4.题目：在组件实例上关于手动侦听事件的三个方法分别是什么？

              答：$on $once $off

              5.题目：有哪两种可以循环使用组件的方式？

              答：组件自身递归使用，两个组件相互引用

              6.题目：使用组件实例的什么方法可以让组件强制更新？

              答：$forceUpdate

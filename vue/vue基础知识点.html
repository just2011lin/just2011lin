                        # vue必背知识点（2.x版本）

                        # 安装

                        1.题目：vue支持哪些浏览器？

                        答：支持所有兼容ECMAScript5的浏览器，不包括IE8及以下版本

                        2.题目：浏览器上安装什么插件可以辅助开发vue应用？

                        答：Vue Devtools

                        3.题目：vue的UMD版本可用于哪些场景？

                        答：在浏览器中使用、用作为AMD模块

                        4.题目：commonJs与ES Module版本有什么区别？

                        答：commonJs用于老的打包工具如Browserify或Webpack1，ES Module版本用于现代打包工具如Webpack2+或Rollup

                        5.题目：什么时候可以只使用运行时版本？

                        答：当使用vue-loader或vueify将*.vue文件在构建时预编译成javascript时

                        6.题目：为什么没有压缩后的ES Module版本？

                        答：ES Module版本为打包工具使用的版本，可以通过打包工具来进行压缩

                        # Vue构造函数的参数

                        题目：有哪些参数，并分别为什么意思

                        interface VueConstructorParam {
    // 渲染元素的css选择器（根组件专享）
    el: string;
    // 响应式数据
    data: Object | Function;
    // 计算属性
    computed: Object,
    // 侦听器
    watch: Object,
    // 方法和事件处理程序
    methods: Object,
    // html模板
    template: string;
    // 自定义attribute
    props: string[] | Object,
    // 局部注册组件
    components: Object;
    // 设置Attribute继承
    inheritAttrs: boolean;
    // 定义v-model使用的属性和事件名
    model: Object;
    // 注册局部指令
    directives: Object;
    // 组件名称
    name: string;
    // 设置函数式组件
    functional: boolean;
    // ...各个生命周期钩子
}

                        这些选项同样是组件的选项

                        # 生命周期钩子

                        题目：有哪些生命周期钩子并做了哪些事情

                        1. beforeCreate：Init Events & Lifecycle
                        2. created：injections & reactivity
                        3. beforeMount: render ready
                        4. mounted: rendered
                        5. beforeUpdate: data change
                        6. updated: re-render
                        7. beforeDestroy: vm.$destroy()
                        8. destroyed: teardown watches,child components and event listeners

                        # 模板语法

                        1.题目：插值的语法是什么？

                        答：Mustache即双大括号

                        2.题目：如何渲染原始的HTML字符串？

                        答：使用v-html指令

                        3.题目：对于布尔属性使用v-bind，在值为哪些情况时将不会渲染该属性？

                        答：undefined,null,false

                        4.题目：模板中哪些部分可以使用javascript表达式？

                        答：所有的数据绑定，包括：插值、指令的值、动态参数

                        5.题目：指令的参数是什么？

                        答：指令冒号后跟的字符串为参数，如 v-bind:href=“http://www.baidu.com” 中href为v-bind指令的参数

                        6.题目：指令的动态参数的语法是怎样的？

                        答：语法是指令+冒号+中括号，中括号内为参数的求值表达式

                        7.题目：修饰符的语法是怎么样的？

                        答：指令或有参数时在参数后用点号加上修饰符名称

                        8.题目：v-bind与v-on的缩写分别是什么？

                        答：冒号和at符号，即:和@

                        # 计算属性与侦听器

                        1.题目：计算属性与方法求值有什么区别？

                        答：计算属性的值是基于它们的响应式依赖进行缓存的，只有响应式依赖发生改变才会进行求值，而方法求值将在每次重新渲染的时候执行。

                        2.题目：计算属性中使用setter时的参数是什么？

                        答：计算属性被赋值的值

                        3.题目：侦听器中的方法的参数是什么样的？

                        答：第一个参数为修改后的新值，第二个参数为修改前的值

                        # CSS与Style绑定

                        1.题目：绑定html class的两种语法分别是什么？

                        答：对象语法和数组语法，对象语法中值为true时将键作为元素的类名，数组语法中将数组中每项的值作为元素的类名

                        2.题目：在组件上添加html class时，vue将如何处理这些类名？

                        答：vue将把这些类名添加到组件根元素上

                        3.题目：绑定内联样式的两种语法分别是什么？

                        答：对象语法和数组语法，对象语法中的对象为常用的css对象，数组语法中的每项为对象语法中的对象

                        # 条件渲染

                        1.题目：有哪些条件渲染指令？

                        答：v-if v-else-if v-else v-show

                        2.题目：有一组同级元素需要在某个条件下控制是否显示，可以使用哪个元素加控制语句来实现？

                        答：template元素

                        3.题目：vue会复用已有元素来重新渲染，可以在元素上设置什么属性来使元素不被复用？

                        答：key属性

                        4.题目：为什么说v-if是惰性的？

                        答：因为只有v-if的条件为真时，才会渲染内容

                        5.题目：v-if与v-show应如何选择？

                        答：频繁切换显示的使用v-show，运行条件改变较少的使用v-if

                        6.题目：v-for与v-if一起使用时，谁的优先级较高？

                        答：v-for的优先级高于v-if

                        # 列表渲染

                        1.题目：v-for用于遍历数组并获取索引的语法？

                        答：v-for="(item,index) in/of items"

                        2.题目：v-for用于遍历对象并获取键与索引的语法？

                        答：v-for="(value,key,index) in/of obj"

                        3.题目：为渲染的列表项设置唯一的key属性有什么作用？

                        答：可以使vue能够重用和重新排序现有的元素

                        4.题目：数组的哪些方法会触发视图更新？

                        答：push pop shift unshift splice sort reverse

                        5.题目：将v-for用于整数时，将会有什么效果？

                        答：会将模板重复对应次数。

                        6.题目：v-for可以用于template元素上吗？

                        答：可以。

                        7.题目：元素的is属性有什么作用？

                        答：is属性的作用为将这个元素渲染为属性值所表示的组件。

                        # 事件处理

                        1.题目：绑定事件处理程序的语法？

                        答：v-on:eventName="语句|方法名|方法名(...参数)"

                        2.题目：原始的DOM事件用什么变量名作为参数传入方法中？

                        答：$event

                        3.题目：有哪些常用的事件修饰符，分别是什么作用？

                        答：stop prevent capture self once passive

                        4.题目：有哪些常用的按键修饰符？

                        答：enter tab delete esc space up down left right

                        5.题目：有哪些系统修饰键？

                        答：ctrl alt shift meta

                        6.题目：exact修饰符有什么用？

                        答：主要与系统修饰键组合使用，只有且当按住了设置的系统修饰键时才触发。不设置系统修饰键时表示没有按住任何系统修饰键才会触发。

                        # 表单输入绑定

                        1.题目：v-model指令可以用于哪三个html元素上实现双向数据绑定？

                        答：input select textarea

                        2.题目：v-model指令在text和textarea元素内使用什么属性和事件？

                        答：value属性和input事件

                        3.题目：v-model指令在checkbox和radio元素内使用什么属性和事件？

                        答：checked属性和change事件

                        4.题目：v-model指令在select元素内部使用什么属性和事件？

                        答：value属性和change事件

                        5.题目：使用v-model绑定单个和多个复选框时，它们的值有什么不同？

                        答：绑定单个复选框时值为布尔类型，绑定多个复选框时值为字符串组成的数组，数组中的每项时选中复选框的value值

                        6.题目：使用v-model绑定多个单选框时，它的值是哪个单选框的value属性值？

                        答：选中即checked属性值为true的那个单选框的value属性值

                        7.题目：单选和多选的select元素用v-model绑定的值在数据类型上有什么区别？

                        答：单选的select元素用v-model绑定的值是选中的option的value值或文本值，多选的select用v-model绑定的值为一个数组。

                        8.题目：可以使用哪两个属性设置单个复选框选中和非选中时的值？

                        答：true-value false-value

                        9.题目：怎么将单选按钮的值设置为除字符串以外的其他类型值？

                        答：使用v-bind指令设置value属性的值为其他类型的值。

                        10.题目：lazy修饰符用于v-model指令后会将输入框在input事件后与数据同步改为在哪个事件后进行同步？

                        答：change

                        11.题目：number修饰符用于v-model时会将输入值转为数值，请问其内部是使用的哪个方法？若不能被解析，将会返回什么值？

                        答：内部使用的parseFloat方法，不能解析时将返回原始值

                        12.题目：使用哪个修饰符可以自动将用户输入的首尾空白字符清除？

                        答：trim

                        # 组件基础

                        1.题目：Vue.component方法接收哪两个参数？

                        答：组件名称、组件选项

                        2.题目：为什么组件的data选项要设置为一个返回响应式数据的方法？

                        答：这样可以使每个组件实例的响应式数据是独立的。

                        3.题目：组件可以有多个根元素吗？

                        答：不行，只能有一个根元素。

                        4.题目：在子组件上通过内建的什么方法来触发一个事件？

                        答：$emit

                        5.题目：子组件上在$emit方法上传入的第二个参数，可以在父组件的模板中用什么字符接收？

                        答：$event

                        6.题目：创建支持v-model指令的组件要实现哪两个内容？

                        答：设置自定义value属性，触发input事件并传入第二个参数作为值

                        7.题目：组件中的插槽的作用是什么？

                        答：将定义于组件元素内的内容分发到组件内部

                        8.题目：在给元素设置v-bind:is属性使，属性值可以是什么内容？

                        答：已注册组件的名称或一个组件的选项对象

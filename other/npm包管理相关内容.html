      # npm包管理相关内容

      # 镜像地址

      源地址：https://registry.npmjs.org/

      淘宝地址：https://registry.npm.taobao.org/

      # 查看与设置镜像地址

      查看：npm config get registry

      设置：npm config set registry ${source_url}

      # 使用yarn

      # 常用命令

      安装：npm install -g yarn

      升级：yarn set version latest

      开启一个新项目(类似npm init)：yarn init

      安装全部npm包：yarn 或 yarn install

      添加包：yarn add [package]

      添加指定版本的包：yarn add [package]@[version]

      添加开发环境依赖的包：yarn add [package] --dev

      升级包：yarn up [package]

      升级包到指定版本：yarn up [package]@[version]

      移除包：yarn remove [package]
